# AI Sales Agent (Shared Hosting Ready)

## Requirements
- PHP 8.1+
- MySQL 5.7+
- Apache with mod_rewrite

## Setup
1. Create MySQL database.
2. Import `database/migrations/schema.sql`.
3. Copy project files to hosting root.
4. Update DB settings in `config/config.php`.
5. Make sure `storage/uploads` is writable.
6. Open the domain in browser.
